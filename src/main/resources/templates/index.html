Welcome to PetCare

Διαχειρίσου κατοικίδια, κλείσε ραντεβού και κράτα ιστορικό επισκέψεων. Οι κτηνίατροι δηλώνουν διαθεσιμότητα και ενημερώνουν σημειώσεις θεραπείας.

Tip: Μετά το login θα γίνει αυτόματο redirect στο dashboard ανάλογα με τον ρόλο (OWNER / VET).

What you can do

Owner
  • Register pets
  • Book appointments
  • View medical/vaccine history
Vet
  • Manage availability
  • Confirm / cancel / complete appointments
  • Record visit notes
Το REST API είναι διαθέσιμο μέσω Swagger και προστατεύεται με JWT (Authorize → Bearer token).
