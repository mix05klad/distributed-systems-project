
# PetCare – Online Pet Management & Appointment System

Το **PetCare** είναι μια διαδικτυακή εφαρμογή διαχείρισης κατοικιδίων και ραντεβού με κτηνίατρους.  
Αναπτύχθηκε στο πλαίσιο του μαθήματος *Κατανεμημένα Συστήματα & Υπηρεσίες* και υλοποιεί βασική λειτουργικότητα για ιδιοκτήτες κατοικιδίων και κτηνίατρους, με στόχο την οργάνωση και παρακολούθηση επισκέψεων, ιστορικού και ειδοποιήσεων.

---

## Λειτουργίες της εφαρμογής

### Για Pet Owners
- Εγγραφή και σύνδεση στο σύστημα  
- Καταχώρηση κατοικιδίων  
- Προβολή λίστας κατοικιδίων  
- Προγραμματισμός ραντεβού  
- Προβολή ιστορικού ιατρικών επισκέψεων  
- Λήψη ειδοποιήσεων (υπό σχεδιασμό)

### Για Vets
- Προβολή εκκρεμών ραντεβού  
- Έγκριση ή απόρριψη ραντεβού  
- Καταχώρηση σημειώσεων επίσκεψης  
- Διαχείριση διαθεσιμότητας (μελλοντική επέκταση)

### Εξωτερική Υπηρεσία Ειδοποιήσεων
Το σύστημα σχεδιάζεται να συνδέεται με εξωτερική REST υπηρεσία για:
- Ειδοποιήσεις ραντεβού  
- Υπενθυμίσεις εμβολιασμού  
- Logging αποστολής ειδοποιήσεων  

---

## Τεχνολογίες

- **Java 21**
- **Spring Boot**
- **Spring Security (JWT Authentication)**
- **Spring Data JPA**
- **H2 Database**
- **Thymeleaf (UI Templates)**
- **REST API Controllers**

## Τρόπος Εκτέλεσης

###  Κάνε Clone το Repository
```bash
git clone https://github.com/mix05klad/distributed-systems-project
cd distributed-systems-project
```
### Τρέξε την εφαρμογή
```bash
./mvnw spring-boot:run
```
### Πρόσβαση στο σύστημα
Μετά την εκκίνηση, η εφαρμογή τρέχει στο:
http://localhost:8080
### Πρόσβαση στην βάση
http://localhost:8080/h2-console
(Credentials βρίσκονται στο `application.properties`)
